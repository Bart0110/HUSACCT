Download and Run HUSACCT

1. Select: Download HUSACCT_x.x JAR File.
Save the jar in a suitable directory.

2. Open/Start the HUSACCT jar
a. The tool starts when you open or double click the jar-file (if your local Java-settings are ok).

b. If not (and you are running Windows), create a shortcut and edit Target to:
java -jar HUSACCT_3.x.jar
For example: "C:\Program Files (x86)\Java\jre7\bin\java.exe" -jar C:\Tools\HUSACCT\HUSACCT_3.2.jar

c. Alternatively, create a .bat file to start HUSACCT. For example, to start HUSACT_3.2:
Create a txt file in the same dir as HUSACCT and type: java -jar HUSACCT_3.2.jar
Save this file as: HUSACCT_3.2Start.bat
Double click this .bat file will start up HUSACCT.

d. Alternatively, start a Command prompt.
Go to the directory with the HUSACCT jar and start the tool from the command line.
For example: C:\Tools>java -jar HUSACCT_3.2.jar.
Note: HUSACCT 3.x requires Java 1.7. If needed, check the Java version: java –version.
