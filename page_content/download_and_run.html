Download and Run

1. Select: Download HUSACCT_x.x JAR File.
Save the jar in a suitable directory.

2. Open/Start the HUSACCT jar
a. The tool starts when you open or double click the jar-file (if your local Java-settings are ok).

b. If not (and you are running Windows), create a shortcut and edit Target to:
java -jar HUSACCT_4.x.jar
For example: "C:\Program Files (x86)\Java\jre7\bin\java.exe" -jar C:\Tools\HUSACCT\HUSACCT_4.0.jar

c. Alternatively, create a .bat file to start HUSACCT. For example, to start HUSACT_4.0:
Create a txt file in the same dir as HUSACCT and type: java -jar HUSACCT_4.0.jar
Save this file as: HUSACCT_4.0Start.bat
Double click this .bat file will start up HUSACCT.

d. Alternatively, start a Command prompt.
Go to the directory with the HUSACCT jar and start the tool from the command line.
For example: C:\Tools>java -jar HUSACCT_4.0.jar.

Note 1: HUSACCT 4.x requires Java 1.8. If needed, check the Java version: java –version.

Note 2: In case of large systems, you can get an OutOfMemoryError with the message “Java heap space”. With the -Xmx JVM argument, you can set the heap size. For instance, allow the JVM to use 1 GB (1024 MB) of memory. E.g. with the following command in a (Windows) bat file:
java -jar -Xmx1024m HUSACCT_4.0.jar.
